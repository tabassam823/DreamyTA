# 1. Italicize Bahasa Serapan
Kata/frasa asing yang kini dibungkus `\emph{}` di seluruh file:

| Term | Contoh hasil |
|------|-------------|
| `ground state` | *ground state* |
| `ansatz` | *ansatz* |
| `entanglement` | *entanglement* |
| `qubit` / `qubits` | *qubit* |
| `trial state` | *trial state* |
| `layers` / `layer` | *layers* |
| `optimizer` | *optimizer* |
| `hardware` | *hardware* |
| `noise` | *noise* |
| `bitstring` | *bitstring* |
| `post-selection` | *post-selection* |
| `NISQ` | *NISQ* |
| `Machine Learning` | *Machine Learning* |
| `Parameter Shift Rule` | *Parameter Shift Rule* (di konteks teks, selain yang sudah `\textit`) |
